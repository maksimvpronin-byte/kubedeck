# KubeDeck 2.7.0

- Node-only baseline remains unchanged: Node 50 / Python 0.
- Resource details can stay open in up to 10 workspace tabs.
- Up to 5 terminal sessions stay connected in a tabbed panel below the resource list.
- Unsaved YAML is protected before workspace navigation.
- Resource tabs retain cluster, namespace, inner drawer tab, and unavailable/not-found state.
- Removing a cluster reports and closes only its related resource and terminal tabs.
