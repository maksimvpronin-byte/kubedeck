# KubeDeck 2.7.5

- Node-only runtime remains active: Node 51 / Python 0.
- Manifest Compare now uses the themed KubeDeck resource chooser instead of a native operating-system select.
- Compare targets show resource context, support keyboard navigation, and ignore stale manifest responses after a newer selection.
- ResourceQuota Summary keeps long resource names separate from progress bars and adapts its rows to narrow drawers.
